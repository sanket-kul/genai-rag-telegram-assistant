# Tech FAQ

## What is Docker?
Docker is a platform used to build, package, and run applications in containers.

## What is Kubernetes?
Kubernetes is an orchestration tool used to manage containerized applications at scale.

## What is an API?
An API (Application Programming Interface) allows different software systems to communicate.

## What is RAG?
RAG (Retrieval-Augmented Generation) combines information retrieval with language models to generate accurate responses.

## What is FastAPI?
FastAPI is a modern Python web framework used to build APIs quickly and efficiently.
